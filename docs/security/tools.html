ELK Stack

SIEM Open Source

Open-source log aggregation and analysis platform (Elasticsearch, Logstash, Kibana).

Distributed search
Data processing pipelines
Real-time visualization
Rich ecosystem
High performance
Cost-effective

Setup Commands:

# Install ELK stack with Docker
docker network create elastic
docker run -d --name elasticsearch --net elastic -p 9200:9200 -p 9300:9300 \
  -e "discovery.type=single-node" elasticsearch:8.5.0
docker run -d --name logstash --net elastic -p 5044:5044 logstash:8.5.0
docker run -d --name kibana --net elastic -p 5601:5601 \
  -e "ELASTICSEARCH_HOSTS=http://elasticsearch:9200" kibana:8.5.0

Common Filters:

# Capture HTTP traffic
http

# Capture specific IP
ip.src == 192.168.1.1

# Capture DNS queries
dns

# Capture TCP SYN packets
tcp.flags.syn == 1

# Capture large packets
frame.len > 1500

Cloud Security Tools

Checkov

IaC Scanner Open Source

Static analysis tool for infrastructure as code security scanning.

Terraform scanning
CloudFormation support
Security best practices
CI/CD integration
Compliance reporting
Custom policies

Usage:

# Scan Terraform files
checkov -f main.tf

# Scan directory
checkov -d terraform/

# CI/CD integration
checkov --framework terraform --output cli

Trivy

Vulnerability Scanner Open Source

Comprehensive vulnerability scanner for containers and artifacts.

Container scanning
Filesystem scanning
Git repository scanning
CI/CD integration
SBOM generation
Fast scanning

Usage:

# Scan container image
trivy image nginx:latest

# Scan filesystem
trivy fs /path/to/project

# Generate SBOM
trivy image --format spdx-json nginx:latest > sbom.json
